# 🚑 RoadSoS

## AI-Powered Emergency Response & Roadside Assistance System

### 🏆 Overview

RoadSoS is an intelligent emergency response platform designed to assist users during accidents and roadside emergencies. The system provides quick access to emergency services, AI-powered guidance, location tracking, accident severity prediction, and SOS alerts to improve response time during critical situations.

---

## ✨ Key Features

### 🚨 Emergency Services

* Ambulance Service (108)
* Police Service (100)
* Fire Brigade (101)
* National Emergency Helpline (112)

### 🤖 AI Emergency Assistant

* Emergency guidance chatbot
* Quick response recommendations
* Voice input support

### 📊 Accident Severity Predictor

* Predicts accident risk based on vehicle speed
* Low Risk
* Medium Risk
* High Risk

### 📍 Live Location Detection

* Detects user's current location
* Displays latitude and longitude coordinates

### 🗺️ Emergency Map

* Interactive map using OpenStreetMap
* Visual support for emergency navigation

### 🚨 SOS Alert System

* One-click emergency alert generation
* Designed for rapid emergency response

### 📄 Emergency Report

* Generates printable emergency information
